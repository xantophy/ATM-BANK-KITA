ATM BANK KITA adalah perangkat lunak simulasi mesin ATM yang memungkinkan pengguna untuk melakukan beberapa tindakan dasar seperti pengecekan saldo, penarikan uang, dan setoran uang. Aplikasi ini akan meminta pengguna untuk memasukkan nomor PIN mereka, kemudian menampilkan menu yang memungkinkan mereka memilih operasi yang ingin dilakukan. Setelah memilih operasi, pengguna dapat memasukkan jumlah uang yang ingin ditarik atau disetor, atau hanya memeriksa saldo mereka.
